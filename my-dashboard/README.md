# My Dashboard

A simple static dashboard with a sidebar, top navbar, and stat cards.

## Preview

Open `index.html` in your browser, or serve the folder locally:

```bash
npx serve .
```

## Structure

```
my-dashboard/
├── index.html      # Main page
├── css/style.css   # Layout and styling
├── js/main.js      # Sidebar toggle
└── README.md
```

## Features

- Fixed sidebar with navigation links
- Sticky top navbar with user menu
- Three responsive stat cards
- Mobile-friendly sidebar toggle
